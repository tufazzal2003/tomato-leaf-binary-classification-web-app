# Tomato Leaf Disease Detection Web App

A Flask-based web application for binary classification of tomato leaf (Healthy
vs Unhealthy) using deep learning.

## Features

- 🌿 Upload tomato leaf images for analysis
- 🤖 AI-powered classification (Healthy/Unhealthy)
- 📊 Confidence score display
- 🎨 Modern, responsive user interface
- 📱 Mobile-friendly design
- ⚡ Real-time predictions
